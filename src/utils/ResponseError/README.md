# ResponseError  

Custom exception to include fetch response.

## Constructor  

**<code>new ResponseError(message: string, resp: Response): ResponseError</code>**  

Create new instance with given message and response.

### Parameters  
- **`message`** <code>string</code> required  
- **`resp`** <code>Response</code> required

## Properties

### response  

Response data.  

**Type:** <code>Response</code>
